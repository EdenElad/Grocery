# ⚠️ הודעה חשובה לגבי הנתונים

## מצב הנתונים הנוכחי

הנתונים הקיימים באפליקציה הם **דמו בלבד** ולא אמינים למטרות שימוש אמיתיות.

### 🔴 בעיות מזוהות:

1. **כתובות פיקטיביות** - "רחוב ראשי" ומספרים מזוייפים
2. **חנויות לא קיימות** - חלק מהחנויות לא קיימות במציאות
3. **ערים חסרות** - חסרים מאות יישובים, כפרים, קיבוצים ומושבים
4. **מחירים לא מעודכנים** - נתונים סטטיים ללא עדכון

---

## ✅ פתרונות מומלצים

### 1. שימוש במקורות נתונים רשמיים

**שקיפות מחירי מזון - משרד הכלכלה והתעשייה:**
- אתר: https://www.gov.il/he/departments/legalInfo/cpfta_prices_regulations
- רשתות השיווק מחויבות לפרסם מחירונים
- נתונים מעודכנים יומית

**API זמינים:**
- Shufersal API (אם קיים רישיון)
- Rami Levy - מחירונים פומביים
- Gov.il - נתוני שקיפות מחירים

### 2. מסד נתונים של חנויות אמיתיות

**מקורות מומלצים:**
- אתרי הרשתות הרשמיים - מפת סניפים
- Google Maps API - לאימות כתובות
- רישום עסקים - משרד הפנים

### 3. רשימת ערים מלאה

**יש להשתמש ב:**
- רשימת רשויות מקומיות - משרד הפנים
- CBS - הלשכה המרכזית לסטטיסטיקה
- כולל: ערים, יישובים, כפרים, קיבוצים, מושבים, התנחלויות
