# 🛒 השוואת מחירי מזון - ישראל

אפליקציית React Native להשוואת מחירי מוצרי מזון בין חנויות שונות בישראל.

## ✨ תכונות

- 📱 עובדת על Android ו-iOS
- 🎨 עיצוב פרימיום עם גרדיאנטים
- 🇮🇱 ממשק בעברית עם תמיכה מלאה ב-RTL
- 🏪 השוואת מחירים בין חנויות שונות
- 📍 סינון לפי אזור ועיר
- 🖼️ תמונות מוצרים
- 🛍️ עגלת קניות חכמה
- 💰 חישוב אוטומטי של הסכום הכולל

## 🚀 התקנה

```bash
# התקן את התלויות
npm install

# הפעל את האפליקציה
npx expo start

# הפעל על אנדרואיד
npx expo start --android

# הפעל על iOS
npx expo start --ios

# הפעל בדפדפן
npx expo start --web
```

## 📲 הרצה על מכשיר

1. הורד את **Expo Go** מה-Play Store (Android) או App Store (iOS)
2. הרץ `npx expo start`
3. סרוק את ה-QR code שמופיע או הזן את הכתובת ידנית
